BENIGN WSBS
1. 01_minimal_safe.wsb (LOW)
   No MappedFolders, Networking disabled, No LogonCommand

2. 02_mapping_safe.wsb (LOW/MEDIUM)
   Map a non‑sensitive folder, ReadOnly=true and No LogonCommand

3) 03_rw_user_profile.wsb 
   Map C:\Users\<YourUser> with ReadOnly=false, Networking enabled

4) 04_rw_logoncommand_script.wsb
   writeable mapping + LogonCommand (batch/PS)

5) 05_apt_style_exfil.wsb
   Heavy writeable mapping + scripted tool download, similar to HTB example (writeable mapping to a “tools” folder + LogonCommand that runs a script which (in a real attack) could download tools / exfiltrate data)

6) 06_malformed.wsb
   malformed XML / robustness test

7) 07_empty.wsb
   empty wsb file
